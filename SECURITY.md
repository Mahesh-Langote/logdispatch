# Security Policy

## Supported Versions

Currently, only the latest major release branch receives security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

We take the security of LogDispatch very seriously. 
If you discover a vulnerability, **please do not open a public issue.**

Instead, please email **maheshlangote777@gmail.com** directly. 

We will acknowledge your email within 48 hours, and we will send a more detailed response within 72 hours indicating the next steps in handling your report. 
After the initial reply to your report, we will keep you informed of the progress towards a fix and full announcement, and may ask for additional information or guidance.
